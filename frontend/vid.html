Visual Interface Design

Voor het vak Visual Interface Design moest ik een nieuwe user interface bedenken en ontwerpen voor het zoeken van een of meerdere boeken uit de schoolbibliotheek, voor scholieren uit VO onderwijs. We moesten ontwerpen voor tablet formaat en moesten het goed laten aansluiten op de doelgroep.

Het proces

Ik vond dit vak erg interessant omdat ik zag dat je doormiddel van veel feedback ook op een heel ander ontwerp kan komen. Mijn eerste design verschilt daarom ook erg met mijn laatste design. Mijn eerste design wilde ik te veel laten lijken op NBD Biblion. Dat was ook de opdrachtgever. Mijn taak was om het ontwerp goed te laten aansluiten op doelgroep door veel onderzoek te doen en heb hier een ontwerp van gemaakt.
